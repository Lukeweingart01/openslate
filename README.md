# openslate
A decentralized browser-based ledger crypto thingie

**openslate** is an experimental, fully decentralized, browser-based ledger system. It uses WebRTC for peer-to-peer swarm communication and SHA-256 Proof of Work consensus to mint the native `SLATE` asset directly inside your web browser.

## 🌐 The Browser App
The core application runs entirely client-side with zero centralized databases or servers.
* **Ephemeral Peer Mesh:** Uses `PeerJS` to automatically discover and connect to other active browser tabs over a WebRTC swarm.
* **Mathematical Identity:** Accounts are secured using local, client-generated threshold cryptography (three interlocking safety sequences).
* **Automine Subroutine:** Features a background hash-engine that runs seamlessly when the browser tab is focused, pausing instantly on blur to preserve your device's battery life and processing power.
